Functions

Functions are one of the most important concepts in mathematics. They are a special, well-behaved type of relation that maps inputs to outputs.

What Makes a Relation a Function?

A function $f$ from a set $A$ to a set $B$ is a relation that satisfies one crucial rule: every element in set $A$ must be related to exactly one element in set $B$.

  • Every element in A is used: No element in the starting set can be left out.
  • Exactly one output for each input: An element in A cannot be mapped to two or more different elements in B.

The Parts of a Function

We use specific terms to describe the sets involved in a function $f: A \rightarrow B$.

  • The Domain is the set of all possible inputs (Set A).
  • The Co-domain is the set of all possible outputs (Set B).
  • The Range is the set of all the actual outputs the function produces. The range is always a subset of the co-domain.

Types of Functions

We classify functions based on how their inputs and outputs are mapped.

  • Injective (One-to-One): Every distinct input maps to a distinct output. No two inputs share the same output.
  • Surjective (Onto): Every element in the co-domain is an output for at least one input. The range is equal to the co-domain.
  • Bijective: The function is both injective and surjective. There is a perfect one-to-one correspondence between the elements of the domain and the co-domain.
