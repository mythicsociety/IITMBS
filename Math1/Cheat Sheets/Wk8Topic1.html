Week 8 – Mathematics 1 Recap (Merged)

1. Continuity at a Point
  • A function is continuous at x = a if:
  • → f(a) is defined
  • → Limit exists at a
  • → f(a) = limit
  • Graph should be unbroken at that point
  • Types of discontinuities: removable, jump, infinite
  • Sandwich (Squeeze) Theorem helps find tricky limits
2. Differentiability
  • Differentiable ⇒ Continuous
  • But continuous does NOT always mean differentiable
  • Not differentiable at corners, jumps, or vertical slopes
  • Functions like |x| and x|x| are continuous but not differentiable at x = 0
  • Graph must be smooth and unbroken for differentiability
3. Linear Approximation
  • Used to estimate function values near a point using tangent line
  • Formula: f(x) ≈ f(a) + f'(a)(x - a)
  • Good for estimating square roots, exponentials, trigonometric values near known points
4. Power Rule
  • Used to differentiate x raised to any power
  • Formula: d/dx(xⁿ) = n·xⁿ⁻¹
  • Works even if n is negative or a fraction
5. Standard Derivatives
  • d/dx(eˣ) = eˣ
  • d/dx(ln(x)) = 1/x
  • d/dx(sin(x)) = cos(x)
  • d/dx(cos(x)) = -sin(x)
  • Useful to memorize these for fast differentiation
6. Product Rule
  • Use when two functions are multiplied
  • Formula: d/dx(u·v) = u'·v + u·v'
  • Example: d/dx(x²·sin(x)) = 2x·sin(x) + x²·cos(x)
7. Quotient Rule
  • Use when one function is divided by another
  • Formula: d/dx(u/v) = (u'·v - u·v') / v²
  • Example: d/dx(x² / (x + 1)) = (2x(x + 1) - x²·1)/(x + 1)²
8. Chain Rule
  • Use when one function is inside another (composite functions)
  • Formula: d/dx[f(g(x))] = f'(g(x)) · g'(x)
  • Example: d/dx(ln(5x + 2)) = 1/(5x + 2) · 5 = 5/(5x + 2)
9. L’Hôpital’s Rule
  • Used when limit gives 0/0 or ∞/∞
  • Take derivatives of numerator and denominator
  • Keep applying if result is still indeterminate
  • Example: lim x→0 (1 - cos(x))/x² = sin(x)/2x → 1/2
10. Limits (Algebraic & Graphical)
  • Limits describe what value a function approaches near a point
  • Left-hand and right-hand limits may differ (discontinuity)
  • Useful for checking continuity and behavior of graphs
